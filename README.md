# Detailed User Instructions

## Step 1: Install Dependencies

Before running the GUI-based agent, make sure to install all dependencies by executing `pip install -r requirements.txt` in your terminal.

## Step 2: Run the Agent

To run the agent, simply execute `python agent.py` in your terminal. If you want to test the GUI, use `python gui_tester.py` instead.

## Step 3: Troubleshoot Issues

If you encounter any issues during execution, refer to the error logs located in the `logs` directory for more information.
